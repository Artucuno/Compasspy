# Compasspy
Python Library for compass.education

## Installation
```
pip install -U git+https://github.com/Artucuno/Compasspy
```


## Simple Example
```py
from compasspy.client import Compass

client = Compass('<School Subdomain>', '<Authentication Cookie>')
client.login()
```
